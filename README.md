# portal-pro-pages

本仓库用于归档和发布多个独立 HTML 页面，不针对任何单个域名或具体网站。

## 项目简介

portal-pro-pages 是一个静态页面集合仓库。这里存放的 HTML 文件彼此独立，可作为通用页面模板、演示页面、备用页面或归档资料使用。仓库本身不涉及特定网站或服务，仅供页面存储与分发参考。

## 目录说明

```
portal-pro-pages/
├── pages/               # 存放所有 HTML 页面文件
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/              # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── README.md            # 本文件
└── LICENSE              # 许可证文件
```

- `pages/`：核心目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面可能引用的静态资源，按类型分类。

## 页面归档说明

- 每个 HTML 页面均为完整文档，可独立打开和运行。
- 页面间无强制依赖关系，但可能共享 `assets/` 中的公共资源。
- 页面文件名应保持有意义的命名（如 `landing-page.html`、`error-page.html`），以便识别内容用途。
- 归档内容不包含敏感信息、实际用户数据或隐私内容。

## 维护说明

- 如需新增页面，请将 HTML 文件放入 `pages/` 目录，并在必要时更新 `assets/` 中引用的资源。
- 删除或修改页面时，请确保不影响其他页面（除非存在显式引用）。
- 建议在提交说明中简要描述页面用途或变更原因。
- 本仓库不自动部署到任何服务，页面仅供手动浏览或分发使用。
